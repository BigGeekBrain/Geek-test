8. Написать свой планировщик задач. - На странице есть текстовое поле для ввода новой задачи. - При нажатии на "Создать" - задача добавляется на страницу. - При наведении на задачу появляется иконка для удаления задачи со страницы. - Добавить в заголовок планировщика кнопку сортировки задач по названию (1 нажатие - по возрастанию, 2 нажатие - по убыванию, 3 нажатие - в порядке добавления ). - Добавить дополнительный список "Важное". У каждой обычной задачи есть кнопка переноса задачи в этот список. При нажатии на неё, задача удаляется из обычного списка и добавляется в список "Важное". Задачу можно перенести обратно с помощью кнопки в списке "Важное". Задачи из списка "Важное" также можно удалить - Написать блок-схему.
